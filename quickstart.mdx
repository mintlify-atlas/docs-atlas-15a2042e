## Requisitos previos

Antes de comenzar, asegúrese de tener instalado:

- **XAMPP** (incluye Apache 2.4+ y MySQL/MariaDB 10.4+)
- **PHP 8.0** o superior (incluido en XAMPP)
- Un navegador web actualizado (Chrome, Firefox, Edge)

## Instalación en 5 pasos

<Steps>
  <Step title="Descargue el repositorio">
    Clone o descargue el repositorio en la carpeta `htdocs` de XAMPP:

    ```bash
    git clone https://github.com/Irving2223/SISTEMA_BIENES_NACIONALES_UPTAG.git "C:/xampp/htdocs/bienes"
    ```

    O descargue el ZIP desde GitHub y extráigalo en `C:/xampp/htdocs/bienes`.
  </Step>

  <Step title="Inicie los servicios de XAMPP">
    Abra el **Panel de Control de XAMPP** e inicie:
    - **Apache** — servidor web
    - **MySQL** — motor de base de datos
  </Step>

  <Step title="Importe la base de datos">
    Abra su navegador y vaya a `http://localhost/phpmyadmin`. Luego:

    1. Haga clic en **Nueva** para crear una base de datos llamada `bienes_nacionales_uptag`
    2. Seleccione la base de datos recién creada
    3. Haga clic en la pestaña **Importar**
    4. Seleccione el archivo `bd_bienes_nacionales.sql` de la raíz del proyecto
    5. Haga clic en **Continuar**

    O desde la terminal:

    ```bash
    mysql -u root -p bienes_nacionales_uptag < bd_bienes_nacionales.sql
    ```
  </Step>

  <Step title="Configure la conexión a la base de datos">
    Edite el archivo `conexion.php` en la raíz del proyecto:

    ```php conexion.php
    <?php
    $host = "localhost";
    $user = "root";      // Usuario de MySQL
    $pass = "";          // Contraseña de MySQL (vacía en XAMPP por defecto)
    $db   = "bienes_nacionales_uptag";

    $conn = mysqli_connect($host, $user, $pass, $db);
    ?>
    ```

    <Warning>
      En producción, use un usuario de base de datos con privilegios limitados y una contraseña segura. Nunca use `root` en producción.
    </Warning>
  </Step>

  <Step title="Acceda al sistema">
    Abra su navegador y navegue a:

    ```
    http://localhost/bienes/
    ```

    Verá la página de bienvenida institucional. Haga clic en **Iniciar Sesión** e ingrese las credenciales por defecto:

    | Campo | Valor |
    |-------|-------|
    | Cédula | `12345678` |
    | Contraseña | `Admin123!` |

    <Warning>
      Cambie estas credenciales inmediatamente después del primer acceso. Vaya a su **Perfil** para actualizar la contraseña.
    </Warning>
  </Step>
</Steps>

## Verificación del sistema

Una vez dentro, el **panel principal** debe mostrar:

- **Total Bienes**: contador de activos registrados
- **Desincorporados**: bienes dados de baja
- **Responsables**: ubicaciones con responsable asignado
- **Bienes de Hoy**: altas registradas en el día

Si todos los contadores cargan sin error, el sistema está funcionando correctamente.

## Primer bien registrado

Para registrar su primer bien nacional:

<Steps>
  <Step title="Ir a Gestión de Bienes">
    En el menú lateral izquierdo, haga clic en **Gestión de Bienes** → **Registrar Bien**.
  </Step>
  <Step title="Completar el formulario">
    Ingrese los datos requeridos:
    - **Código de Bien Nacional** (o déjelo vacío para generación automática)
    - **Descripción** del bien
    - **Categoría** (ej. Mobiliario y Equipo de Oficina)
    - **Estatus** (ej. Activo)
  </Step>
  <Step title="Guardar">
    Haga clic en **Registrar**. El bien aparecerá en el dashboard en la tabla "Últimos Bienes Registrados".
  </Step>
</Steps>
